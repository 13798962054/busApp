# busApp

+ 这时一款微信小程序，成员类型包括：
  + 司机
  + 乘客
+ 界面包含：
  + 司机页面
    + 发车表
    + 按钮（我要发车），跳转到发车表单页面，该页面包含：
      + 时间
      + 车型（默认上次选用的类型）
      + 出发地（大鹏、先科、盐田）
      + 途径站点
      + 提交按钮
      + 返回按钮
  + 乘客页面
    + 拼车名单
    + 按钮（我要乘车），跳转到乘车表单页面，该页面包含：
      + 时间
      + 出发点
      + 终点
      + 携带东西
      + 确认预定
      + 取消
